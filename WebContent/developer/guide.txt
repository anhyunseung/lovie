## encoding을 UTF-8로 변경하기
1. Window - preferences - General - Content Types - Java Class File / Text의 Default - encoding - UTF-8 - Update
2. Window - preferences - General - Workspace - Text file encoding - UTF-8
3. Window - preferences - Web - CSS Files / HTML Files / JSP Files - Encoding - UTF-8
4. Window - preferences - XML - XML Files - Encoding - UTF-8

ps. 니가 그동안 만들어 놧던 jsp파일들 UTF-8로 해놓은거 다 UTF-8로 고쳐놔 (웬만한 프로젝트의 기본은 UTF-8이야 / ctrl + H 를 써서 검색하면 더 빠름)
